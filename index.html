Programming

At the age of 12, I wrote my first Hello World program in BASIC. Using some references from the internet, I also wrote a simple text-based adventure game in BASIC. Then, at university, I was introduced to C++ and Java. I took a keen interest in C++ and dove deeper into the fundamentals of object-oriented programming. In the beginning, I used C++ for solving mathematical problems such as the ones presented in ProjectEuler.net. However, I also created some simple console games, such as Tic-Tac-Toe, in C++. Using the cumulative programming knowledge, I also created a Java application for my Capstone project. Some of these projects are showcased on my GitHub Profile.

Currently, I am working on creating an AI for the Tic-Tac-Toe game that uses MiniMax algorithm with Alpha-Beta pruning. At the same time, I am also learning about the fundamentals of Machine Learning and Game Design.
